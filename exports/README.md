# Exports

Exported diagram files in various formats (PNG, SVG, PDF).

## Purpose

- Share diagrams in documentation
- Embed in presentations
- Print for meetings
- Email to stakeholders
- Include in reports

## File Formats

- **PNG** - Raster image, good for documentation and web
- **SVG** - Vector image, scalable and web-friendly
- **PDF** - Printable, good for formal documentation

## Naming Convention

Match the source .drawio filename:
- `claude_code_dataflow.png` (from `claude_code_dataflow.drawio`)
- `api_architecture.svg` (from `api_architecture.drawio`)
- `system_design.pdf` (from `system_design.drawio`)

## Export Settings (Draw.io)

**PNG Export:**
- File → Export as → PNG
- Zoom: 100%
- Border Width: 10px
- Transparent Background: Optional

**SVG Export:**
- File → Export as → SVG
- Include copy of diagram: Yes (embeds source)
- Size: Page

**PDF Export:**
- File → Export as → PDF
- Page View: Fit to page width
- Include copy of diagram: Yes
