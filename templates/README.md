# Templates

Reusable diagram templates and starter files.

## Examples

- Blank workflow template
- Architecture diagram template
- ERD template
- Process flow template
- Standard shapes and symbols library

## Usage

1. Copy template file
2. Rename with your diagram name
3. Customize for your needs
4. Save in appropriate folder (workflows, architecture, etc.)

## Naming Convention

- `template_workflow.drawio` - Workflow template
- `template_architecture.drawio` - Architecture template
- `template_erd.drawio` - ERD template
