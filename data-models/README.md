# Data Models

Data model diagrams, database schemas, and entity-relationship diagrams.

## Examples

- Database table schemas
- Entity-relationship diagrams (ERD)
- Dimension structures (OneStream)
- Data flow diagrams
- Cube structures

## Naming Convention

- `tablename_schema.drawio` - Database table schema
- `dimension_structure.drawio` - Dimension hierarchy
- `entity_relationships.drawio` - ERD diagrams
